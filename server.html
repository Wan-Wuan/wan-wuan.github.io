使用帮助

  • ✅ 所有链接请替换为你的实际 IP 或域名（如 http://192.168.1.100:8080）
  • 🔒 对外网开放前，请务必配置 HTTPS 和身份认证（推荐 Authelia 或 Nginx Basic Auth）
  • 🔄 如需自动发现 Docker 服务，可集成 heimdall 或编写脚本读取 Docker API
  • 🎨 主题切换按钮位于右上角，支持暗色/亮色模式

推荐工具栈

  • 反向代理：Nginx Proxy Manager / Traefik
  • 认证网关：Authelia
  • 监控：Prometheus + Grafana + Node Exporter
  • 自动化：Watchtower（自动更新容器）

关于本面板

这是一个轻量级、无依赖的自建服务器首页，专为家庭实验室（Home Lab）或个人 NAS 设计。

✨ 特性：多页面导航、主题切换、响应式布局、交互动画、预留 API 扩展接口。
